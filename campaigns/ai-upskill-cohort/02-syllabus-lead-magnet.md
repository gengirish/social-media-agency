## PAGE 2

### 5 PORTFOLIO-GRADE PROJECTS (with real ROI)

Most bootcamp graduates build chatbots. These solve expensive real-world problems.

**P1 · Compliance Sentinel Agent** — *Saves legal teams 40+ hrs/week* (W05)
Crawls regulatory portals daily, embeds rules into a legal RAG store, and a LangGraph agent audits a codebase against them — producing audit-ready PDF reports with citations.
*Stack: Playwright · Supabase pgvector · LangGraph · BullMQ · WeasyPrint · Slack/email alerts*

**P2 · Autonomous Code Reviewer** — *Cuts code review time 70%* (W06)
On every GitHub PR, 4 specialised agents (Security, Style, Tests, Architecture) analyse the diff in parallel; an LLM-as-judge aggregates a consensus review and posts it to the PR.
*Stack: GitHub webhooks · multi-agent orchestration · AST analysis · LLM-as-judge*

**P3 · Research-to-Deck Generator** — *3 days of research → 5 minutes* (W04)
Pulls 50+ papers via Semantic Scholar, embeds them, runs multi-query RAG to synthesise findings, and exports an editable PPTX with per-claim citations and speaker notes.
*Stack: Semantic Scholar API · multi-document RAG · programmatic PPTX · async jobs*

**P4 · Recruiting Swarm** — *+40% interview show-up rates* (W07)
NER parses resumes into structured profiles, scores fit vs a JD by vector similarity, drafts personalised outreach, and auto-books interviews via Google Calendar — with a recruiter dashboard.
*Stack: NER · vector similarity · multi-step agents · multi-tenant Next.js SaaS*

**P5 · Cost-Aware Agent Router** — *Reduces inference costs 60%* (W10)
A drop-in LLM proxy that classifies query complexity, routes to the cheapest capable model, enforces per-org token budgets, and exposes Prometheus cost metrics.
*Stack: LLM gateway · dynamic model selection · Prisma budgets · Prometheus*
